Detail Kursus: Web Programming

Pelajari pengembangan web dari dasar hingga mahir

Deskripsi Kursus

Kursus Web Programming dirancang untuk memberikan pemahaman menyeluruh tentang pengembangan web modern. Mulai dari fondasi HTML dan CSS, hingga JavaScript interaktif dan framework populer seperti Bootstrap dan React.js.

Kursus ini cocok untuk pemula yang ingin memulai karir sebagai web developer, maupun profesional yang ingin meningkatkan keterampilan mereka.

Durasi

26 Minggu

Level

Pemula - Lanjutan

Peserta

Maksimal 30 orang

Sertifikat

Ya, tersedia

Topik Per Minggu

  • Minggu 1-4
    HTML Fundamentals
    • Struktur dasar HTML dan semantic elements
    • Forms, tables, dan media embedding
    • Best practices dan accessibility
  • Minggu 5-9
    CSS Styling & Layout
    • Selectors, properties, dan box model
    • Flexbox dan Grid layout
    • Responsive design dan animations
  • Minggu 10-15
    JavaScript Interaktif
    • Variables, functions, dan operators
    • DOM manipulation dan event handling
    • Asynchronous JavaScript
  • Minggu 16-18
    Bootstrap Framework
    • Grid system dan components
    • Responsive utilities
    • Customizing dengan Sass
  • Minggu 19-26
    React.js Basics
    • Components, Props, dan JSX
    • State management dan Hooks
    • Final project web application
